# **Python SDK demo 使用文档**


### 第一步：下载SDK
下载后得到的文件目录结构如下:
```
    |
    |___imgResource
    |       |___622747
    |       |___107434
    |             |___T0
    |             |___T1
    |
    |___PythonSDK
    |      |___init.py
    |      |___sigmacloud.py
    |      |___utils.py
    |
    |___call.py
    |___Python SDK demo 使用文档.md
```

### 第二步：打开python开发工具，这里以PyCharm为例：
```
    点击PyCharm的Open，打开pythonSDK,进入pyCharm界面。
```

### 第三步：开始调用API
```
    打开call.py文件，运行其中的逻辑
    （如果鉴权方式用“请求签名”,则填写API_KEY和API_SECRET)
     (如果鉴权方式用“token”，则登录)
```
